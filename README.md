# CRAWLER NEWS DATA FROM WEBSITE SCHOOL
Use NodeJs (framework ExpressJs) width typescript

## Install
```
git clone https://github.com/anhocva214/crawler-news-schools.git
yarn install
```
## Run Development
```
yarn dev
```

## Run Production
```
yarn build && yarn start
```
